# Nagar Shree Churu Website

Official website project for **Lok Sanskriti Shodh Sansthan – Nagar Shree Churu**, Rajasthan.

A modern heritage-inspired website designed to preserve and showcase the cultural, literary, and historical richness of Churu and Rajasthan through an elegant and responsive digital experience.

---

## Features

- Modern responsive UI
- Heritage-inspired design
- Smooth scroll animations
- Gallery section
- Secretary profile section
- Contact information page
- Mobile-friendly layout
- Clean folder structure
- Easy future scalability

---

## Tech Stack

- HTML5
- CSS3
- JavaScript
- AOS Animation Library
- GSAP (planned)

## Project Goal

The goal of this project is to create a visually rich and culturally meaningful digital platform for Nagar Shree Churu that reflects the heritage and traditions of Rajasthan while maintaining a modern web experience.
